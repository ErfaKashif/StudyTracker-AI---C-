# StudyTracker-AI---C-
StudyTracker-CPP is a console-based study management platform developed as a Data Structures project in C++. The application helps students organize study materials, create flashcards, generate quizzes from structured notes, and collaborate through study session chat rooms.

The project focuses on applying core data structures and algorithms to solve real-world educational problems.

---

## Features

### Subject & Study Session Management

* Create and manage study subjects
* Organize study sessions
* Track study activities

### Notes Management

* Create and store study notes
* Organize notes by subject
* Retrieve notes efficiently

### Flashcard System

* Create flashcards for active recall learning
* Review flashcards during study sessions

### Quiz Generation

* Automatically generate quiz questions from notes written in predefined formats
* Reinforce learning through self-assessment

### Student Chat Rooms

* Join study session discussions
* Collaborate with other students
* Share learning resources and ideas

### Console-Based Interface

* Interactive menu-driven navigation
* Lightweight and easy to run
